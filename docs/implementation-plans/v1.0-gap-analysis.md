# v1.0 Feature Gap Analysis: playwright-rust vs playwright-python

**Last Updated:** 2026-04-16

**Baseline:** playwright-rust v0.11.0 vs playwright-python (Playwright v1.50+)

## Goal

Comprehensive API gap analysis comparing this repo's Rust bindings against the official Playwright Python bindings. Organized by priority to guide v1.0 development.

## P0 ÔÇö Critical for v1.0

### Locator: `get_by_*` Methods

These are *the* modern Playwright locator pattern. Without them, users cannot write idiomatic Playwright tests.

- [x] `get_by_role(role)` ÔÇö Locate by ARIA role (most important)
- [x] `get_by_text(text)` ÔÇö Locate by text content
- [x] `get_by_label(text)` ÔÇö Locate by associated label
- [x] `get_by_placeholder(text)` ÔÇö Locate by placeholder text
- [x] `get_by_test_id(test_id)` ÔÇö Locate by test ID attribute
- [x] `get_by_alt_text(text)` ÔÇö Locate by alt text
- [x] `get_by_title(text)` ÔÇö Locate by title attribute

These must also be available on **Page** and **Frame** (they delegate to the main frame's locator).

### Locator: Filtering & Composition

- [x] `filter()` ÔÇö Narrow locator with additional conditions (has_text, has, etc.)
- [x] `and_(locator)` ÔÇö Match both this AND another locator
- [x] `or_(locator)` ÔÇö Match either this OR another locator
- [x] `all()` ÔÇö Return array of locators for each match

### Locator: Missing Interactions

- [x] `blur()` ÔÇö Remove focus
- [x] `focus()` ÔÇö Set focus
- [x] `bounding_box()` ÔÇö Get element dimensions/position
- [x] `drag_to(target)` ÔÇö Drag and drop
- [x] `dispatch_event(type)` ÔÇö Fire DOM events
- [x] `evaluate(expression)` / `evaluate_all(expression)` ÔÇö JS on elements
- [x] `press_sequentially(text)` ÔÇö Character-by-character typing
- [x] `scroll_into_view_if_needed()` ÔÇö Scroll into viewport
- [x] `wait_for()` ÔÇö Wait for element state
- [x] `tap()` ÔÇö Touch gesture
- [x] `is_hidden()` / `is_disabled()` ÔÇö Missing negative state checks
- [x] `all_inner_texts()` / `all_text_contents()` ÔÇö Bulk text retrieval
- [x] `page` property ÔÇö Back-reference to owning Page

### Page: Core Missing Methods

- [x] `go_back()` ÔÇö History back navigation
- [x] `go_forward()` ÔÇö History forward navigation
- [x] `set_content(html)` ÔÇö Set page HTML directly
- [x] `wait_for_load_state(state)` ÔÇö Wait for `load/domcontentloaded/networkidle`
- [x] `wait_for_url(url)` ÔÇö Wait for navigation to matching URL
- [x] `set_default_timeout(timeout)` ÔÇö Default timeout for all operations
- [x] `set_default_navigation_timeout(timeout)` ÔÇö Navigation-specific timeout
- [x] `is_closed` property ÔÇö Whether page has been closed
- [x] `frames` property ÔÇö List all frames (main frame only; iframe enumeration planned)

### Page: `get_by_*` Convenience Methods

- [x] `get_by_role(role)`
- [x] `get_by_text(text)`
- [x] `get_by_label(text)`
- [x] `get_by_placeholder(text)`
- [x] `get_by_test_id(test_id)`
- [x] `get_by_alt_text(text)`
- [x] `get_by_title(text)`
- [x] `frame_locator(selector)` ÔÇö FrameLocator for iframe content

### BrowserContext: Cookie & State Management

- [x] `add_cookies(cookies)` ÔÇö Add cookies
- [x] `cookies(urls)` ÔÇö Get cookies (optionally filtered)
- [x] `clear_cookies()` ÔÇö Remove cookies (with optional filter)
- [x] `route(url, handler)` / `unroute(url)` / `unroute_all()` ÔÇö Context-level routing
- [x] `set_default_timeout(timeout)` / `set_default_navigation_timeout(timeout)`
- [x] `set_extra_http_headers(headers)`
