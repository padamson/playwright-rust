**Last Updated:** 2026-04-16

Comprehensive API gap analysis comparing this repo's Rust bindings against the official Playwright Python bindings. Organized by priority to guide v1.0 development.

## P0 — Critical for v1.0

### Locator: `get_by_*` Methods

These are *the* modern Playwright locator pattern. Without them, users cannot write idiomatic Playwright tests.

- [x] `get_by_role(role)` — Locate by ARIA role (most important)
- [x] `get_by_text(text)` — Locate by text content
- [x] `get_by_label(text)` — Locate by associated label
- [x] `get_by_placeholder(text)` — Locate by placeholder text
- [x] `get_by_test_id(test_id)` — Locate by test ID attribute
- [x] `get_by_alt_text(text)` — Locate by alt text
- [x] `get_by_title(text)` — Locate by title attribute

These must also be available on **Page** and **Frame** (they delegate to the main frame's locator).

### Locator: Filtering & Composition

- [x] `filter()` — Narrow locator with additional conditions (has_text, has, etc.)
- [x] `and_(locator)` — Match both this AND another locator
- [x] `or_(locator)` — Match either this OR another locator
- [x] `all()` — Return array of locators for each match

### Locator: Missing Interactions

- [x] `blur()` — Remove focus
- [x] `focus()` — Set focus
- [x] `bounding_box()` — Get element dimensions/position
- [x] `drag_to(target)` — Drag and drop
- [x] `dispatch_event(type)` — Fire DOM events
- [x] `evaluate(expression)` / `evaluate_all(expression)` — JS on elements
- [x] `press_sequentially(text)` — Character-by-character typing
- [x] `scroll_into_view_if_needed()` — Scroll into viewport
- [x] `wait_for()` — Wait for element state
- [x] `tap()` — Touch gesture
- [x] `is_hidden()` / `is_disabled()` — Missing negative state checks
- [x] `all_inner_texts()` / `all_text_contents()` — Bulk text retrieval
- [x] `page` property — Back-reference to owning Page

### Page: Core Missing Methods

- [x] `go_back()` — History back navigation
- [x] `go_forward()` — History forward navigation
- [x] `set_content(html)` — Set page HTML directly
- [x] `wait_for_load_state(state)` — Wait for `load/domcontentloaded/networkidle`
- [x] `wait_for_url(url)` — Wait for navigation to matching URL
- [x] `set_default_timeout(timeout)` — Default timeout for all operations
- [x] `set_default_navigation_timeout(timeout)` — Navigation-specific timeout
- [x] `is_closed` property — Whether page has been closed
- [x] `frames` property — List all frames (main frame only; iframe enumeration planned)

### Page: `get_by_*` Convenience Methods

- [x] `get_by_role(role)`
- [x] `get_by_text(text)`
- [x] `get_by_label(text)`
- [x] `get_by_placeholder(text)`
- [x] `get_by_test_id(test_id)`
- [x] `get_by_alt_text(text)`
- [x] `get_by_title(text)`
- [x] `frame_locator(selector)` — FrameLocator for iframe content

### BrowserContext: Cookie & State Management

- [x] `add_cookies(cookies)` — Add cookies
- [x] `cookies(urls)` — Get cookies (optionally filtered)
- [x] `clear_cookies()` — Remove cookies (with optional filter)
- [x] `route(url, handler)` / `unroute(url)` / `unroute_all()` — Context-level routing
- [x] `set_default_timeout(timeout)` / `set_default_navigation_timeout(timeout)`
- [x] `set_extra_http_headers(headers)`
- [x] `grant_permissions(permissions)` / `clear_permissions()`
- [x] `set_geolocation(geolocation)` / `set_offline(offline)`
