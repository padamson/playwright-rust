Comprehensive API gap analysis comparing this repo's Rust bindings against the official Playwright Python bindings. Organized by priority to guide v1.0 development.

## P0 тАФ Critical for v1.0

### Locator: `get_by_*` Methods

These are *the* modern Playwright locator pattern. Without them, users cannot write idiomatic Playwright tests.

- [x] `get_by_role(role)` тАФ Locate by ARIA role (most important)
- [x] `get_by_text(text)` тАФ Locate by text content
- [x] `get_by_label(text)` тАФ Locate by associated label
- [x] `get_by_placeholder(text)` тАФ Locate by placeholder text
- [x] `get_by_test_id(test_id)` тАФ Locate by test ID attribute
- [x] `get_by_alt_text(text)` тАФ Locate by alt text
- [x] `get_by_title(text)` тАФ Locate by title attribute

These must also be available on **Page** and **Frame** (they delegate to the main frame's locator).

### Locator: Filtering & Composition

- [x] `filter()` тАФ Narrow locator with additional conditions (has_text, has, etc.)
- [x] `and_(locator)` тАФ Match both this AND another locator
- [x] `or_(locator)` тАФ Match either this OR another locator
- [x] `all()` тАФ Return array of locators for each match

### Locator: Missing Interactions

- [x] `blur()` тАФ Remove focus
- [x] `focus()` тАФ Set focus
- [x] `bounding_box()` тАФ Get element dimensions/position
- [x] `drag_to(target)` тАФ Drag and drop
- [x] `dispatch_event(type)` тАФ Fire DOM events
- [x] `evaluate(expression)` / `evaluate_all(expression)` тАФ JS on elements
- [x] `press_sequentially(text)` тАФ Character-by-character typing
- [x] `scroll_into_view_if_needed()` тАФ Scroll into viewport
- [x] `wait_for()` тАФ Wait for element state
- [x] `tap()` тАФ Touch gesture
- [x] `is_hidden()` / `is_disabled()` тАФ Missing negative state checks
- [x] `all_inner_texts()` / `all_text_contents()` тАФ Bulk text retrieval
- [x] `page` property тАФ Back-reference to owning Page

### Page: Core Missing Methods

- [x] `go_back()` тАФ History back navigation
- [x] `go_forward()` тАФ History forward navigation
- [x] `set_content(html)` тАФ Set page HTML directly
- [x] `wait_for_load_state(state)` тАФ Wait for `load/domcontentloaded/networkidle`
- [x] `wait_for_url(url)` тАФ Wait for navigation to matching URL
- [x] `set_default_timeout(timeout)` тАФ Default timeout for all operations
- [x] `set_default_navigation_timeout(timeout)` тАФ Navigation-specific timeout
- [x] `is_closed` property тАФ Whether page has been closed
- [x] `frames` property тАФ List all frames (main frame only; iframe enumeration planned)

### Page: `get_by_*` Convenience Methods

- [x] `get_by_role(role)`
- [x] `get_by_text(text)`
- [x] `get_by_label(text)`
- [x] `get_by_placeholder(text)`
- [x] `get_by_test_id(test_id)`
- [x] `get_by_alt_text(text)`
- [x] `get_by_title(text)`
- [x] `frame_locator(selector)` тАФ FrameLocator for iframe content

### BrowserContext: Cookie & State Management

- [x] `add_cookies(cookies)` тАФ Add cookies
- [x] `cookies(urls)` тАФ Get cookies (optionally filtered)
- [x] `clear_cookies()` тАФ Remove cookies (with optional filter)
- [x] `route(url, handler)` / `unroute(url)` / `unroute_all()` тАФ Context-level routing
- [x] `set_default_timeout(timeout)` / `set_default_navigation_timeout(timeout)`
- [x] `set_extra_http_headers(headers)`
- [x] `grant_permissions(permissions)` / `clear_permissions()`
