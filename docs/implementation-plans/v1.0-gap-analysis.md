# v1.0 Feature Gap Analysis: playwright-rust vs playwright-python

**Last Updated:** 2026-04-16

**Baseline:** playwright-rust v0.11.0 vs playwright-python (Playwright v1.50+)

## Goal

Comprehensive API gap analysis comparing this repo's Rust bindings against the official Playwright Python bindings. Organized by priority to guide v1.0 development.

## P0 ‚Äî Critical for v1.0

### Locator: `get_by_*` Methods

These are *the* modern Playwright locator pattern. Without them, users cannot write idiomatic Playwright tests.

- [x] `get_by_role(role)` ‚Äî Locate by ARIA role (most important)
- [x] `get_by_text(text)` ‚Äî Locate by text content
- [x] `get_by_label(text)` ‚Äî Locate by associated label
- [x] `get_by_placeholder(text)` ‚Äî Locate by placeholder text
- [x] `get_by_test_id(test_id)` ‚Äî Locate by test ID attribute
- [x] `get_by_alt_text(text)` ‚Äî Locate by alt text
- [x] `get_by_title(text)` ‚Äî Locate by title attribute

These must also be available on **Page** and **Frame** (they delegate to the main frame's locator).

### Locator: Filtering & Composition

- [x] `filter()` ‚Äî Narrow locator with additional conditions (has_text, has, etc.)
- [x] `and_(locator)` ‚Äî Match both this AND another locator
- [x] `or_(locator)` ‚Äî Match either this OR another locator
- [x] `all()` ‚Äî Return array of locators for each match

### Locator: Missing Interactions

- [x] `blur()` ‚Äî Remove focus
- [x] `focus()` ‚Äî Set focus
- [x] `bounding_box()` ‚Äî Get element dimensions/position
- [x] `drag_to(target)` ‚Äî Drag and drop
- [x] `dispatch_event(type)` ‚Äî Fire DOM events
- [x] `evaluate(expression)` / `evaluate_all(expression)` ‚Äî JS on elements
- [x] `press_sequentially(text)` ‚Äî Character-by-character typing
- [x] `scroll_into_view_if_needed()` ‚Äî Scroll into viewport
- [x] `wait_for()` ‚Äî Wait for element state
- [x] `tap()` ‚Äî Touch gesture
- [x] `is_hidden()` / `is_disabled()` ‚Äî Missing negative state checks
- [x] `all_inner_texts()` / `all_text_contents()` ‚Äî Bulk text retrieval
- [x] `page` property ‚Äî Back-reference to owning Page

### Page: Core Missing Methods

- [x] `go_back()` ‚Äî History back navigation
- [x] `go_forward()` ‚Äî History forward navigation
- [x] `set_content(html)` ‚Äî Set page HTML directly
- [x] `wait_for_load_state(state)` ‚Äî Wait for `load/domcontentloaded/networkidle`
- [x] `wait_for_url(url)` ‚Äî Wait for navigation to matching URL
- [x] `set_default_timeout(timeout)` ‚Äî Default timeout for all operations
- [x] `set_default_navigation_timeout(timeout)` ‚Äî Navigation-specific timeout
- [x] `is_closed` property ‚Äî Whether page has been closed
- [x] `frames` property ‚Äî List all frames (main frame only; iframe enumeration planned)

### Page: `get_by_*` Convenience Methods

- [x] `get_by_role(role)`
- [x] `get_by_text(text)`
- [x] `get_by_label(text)`
- [x] `get_by_placeholder(text)`
- [x] `get_by_test_id(test_id)`
- [x] `get_by_alt_text(text)`
- [x] `get_by_title(text)`
- [x] `frame_locator(selector)` ‚Äî FrameLocator for iframe content

### BrowserContext: Cookie & State Management

- [x] `add_cookies(cookies)` ‚Äî Add cookies
- [x] `cookies(urls)` ‚Äî Get cookies (optionally filtered)
- [x] `clear_cookies()` ‚Äî Remove cookies (with optional filter)
- [x] `route(url, handler)` / `unroute(url)` / `unroute_all()` ‚Äî Context-level routing
- [x] `set_default_timeout(timeout)` / `set_default_navigation_timeout(timeout)`
- [x] `set_extra_http_headers(headers)`
- [x] `grant_permissions(permissions)` / `clear_permissions()`
