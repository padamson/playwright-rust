## P0 — Critical for v1.0

### Locator: `get_by_*` Methods

These are *the* modern Playwright locator pattern. Without them, users cannot write idiomatic Playwright tests.

- [x] `get_by_role(role)` — Locate by ARIA role (most important)
- [x] `get_by_text(text)` — Locate by text content
- [x] `get_by_label(text)` — Locate by associated label
- [x] `get_by_placeholder(text)` — Locate by placeholder text
- [x] `get_by_test_id(test_id)` — Locate by test ID attribute
- [x] `get_by_alt_text(text)` — Locate by alt text
- [x] `get_by_title(text)` — Locate by title attribute

These must also be available on **Page** and **Frame** (they delegate to the main frame's locator).

### Locator: Filtering & Composition

- [x] `filter()` — Narrow locator with additional conditions (has_text, has, etc.)
- [x] `and_(locator)` — Match both this AND another locator
- [x] `or_(locator)` — Match either this OR another locator
- [x] `all()` — Return array of locators for each match

### Locator: Missing Interactions

- [x] `blur()` — Remove focus
- [x] `focus()` — Set focus
- [x] `bounding_box()` — Get element dimensions/position
- [x] `drag_to(target)` — Drag and drop
- [x] `dispatch_event(type)` — Fire DOM events
- [x] `evaluate(expression)` / `evaluate_all(expression)` — JS on elements
- [x] `press_sequentially(text)` — Character-by-character typing
- [x] `scroll_into_view_if_needed()` — Scroll into viewport
- [x] `wait_for()` — Wait for element state
- [x] `tap()` — Touch gesture
- [x] `is_hidden()` / `is_disabled()` — Missing negative state checks
- [x] `all_inner_texts()` / `all_text_contents()` — Bulk text retrieval
- [x] `page` property — Back-reference to owning Page

### Page: Core Missing Methods

- [x] `go_back()` — History back navigation
- [x] `go_forward()` — History forward navigation
- [x] `set_content(html)` — Set page HTML directly
- [x] `wait_for_load_state(state)` — Wait for `load/domcontentloaded/networkidle`
- [x] `wait_for_url(url)` — Wait for navigation to matching URL
- [x] `set_default_timeout(timeout)` — Default timeout for all operations
- [x] `set_default_navigation_timeout(timeout)` — Navigation-specific timeout
- [x] `is_closed` property — Whether page has been closed
- [x] `frames` property — List all frames (main frame only; iframe enumeration planned)

### Page: `get_by_*` Convenience Methods

- [x] `get_by_role(role)`
- [x] `get_by_text(text)`
- [x] `get_by_label(text)`
- [x] `get_by_placeholder(text)`
- [x] `get_by_test_id(test_id)`
- [x] `get_by_alt_text(text)`
- [x] `get_by_title(text)`
- [ ] `frame_locator(selector)` — FrameLocator for iframe content

### BrowserContext: Cookie & State Management

- [x] `add_cookies(cookies)` — Add cookies
- [x] `cookies(urls)` — Get cookies (optionally filtered)
- [x] `clear_cookies()` — Remove cookies (with optional filter)
- [x] `route(url, handler)` / `unroute(url)` / `unroute_all()` — Context-level routing
- [x] `set_default_timeout(timeout)` / `set_default_navigation_timeout(timeout)`
- [x] `set_extra_http_headers(headers)`
- [x] `grant_permissions(permissions)` / `clear_permissions()`
- [x] `set_geolocation(geolocation)` / `set_offline(offline)`

## P1 — High Priority

### Response: Body Access

Currently Response is a simple struct with no body access. Users need:

- [x] `body()` — Response body as bytes
- [x] `text()` — Response body as text
- [x] `json()` — Parsed JSON body
- [x] `finished()` — Wait for response to finish loading
- [x] `all_headers()` / `header_value(name)` / `headers_array()`
- [x] `security_details()` / `server_addr()`
- [x] `request` property — Back-reference to Request
- [x] `frame` property

### Request: Missing Properties & Methods

- [x] `headers` — Request headers dict (from initializer, returns HashMap)
- [x] `post_data` / `post_data_buffer` / `post_data_json` — Request body (base64-decoded)
- [x] `failure` — Error info if request failed (from `requestFailed` event `failureText`)
- [x] `all_headers()` / `header_value(name)` / `headers_array()` — Full raw headers via `rawRequestHeaders` RPC
- [x] `timing` — Resource timing info (from Response initializer on `requestFinished`)
- [x] `frame` — Owning frame
- [x] `redirected_from` / `redirected_to` — Redirect chain
- [x] `response()` — Get matching Response
- [x] `sizes()` — Resource size info (requires `response()`)

**Note:** `postData` in the Playwright protocol is base64-encoded. `timing()` data lives in the
Response initializer and is extracted when `requestFinished` fires in `dispatch_request_event`.

### Route: ~~Missing Methods~~ Complete

- [x] `fallback(overrides)` — Continue but allow other handlers to run
- [x] `fetch(overrides)` — Perform request, return response for modification (via APIRequestContext)

### FrameLocator — Entire Class Missing

Required for testing apps with iframes:

- [ ] `FrameLocator` struct
- [ ] `locator(selector)` — Create locator within frame
- [ ] `frame_locator(selector)` — Nested FrameLocator
- [ ] `get_by_role/text/label/placeholder/test_id/alt_text/title` — All `get_by_*` methods
- [ ] `owner` property — Locator for the iframe element

### BrowserType: CDP Connection (Issue #32)

Required for connecting to remote browsers (browserless, Chrome DevTools, etc.):

- [x] `connect_over_cdp(endpoint_url)` — Connect to a Chrome DevTools Protocol endpoint

### Frame: Undercovered

- [ ] `get_by_*` methods (all 7)
- [ ] `locator(selector)` — Create Locator (currently internal only)
- [ ] `frame_locator(selector)` — FrameLocator for nested iframes
- [ ] `set_content(html)` — Set frame HTML
- [ ] `evaluate(expression)` / `evaluate_handle(expression)`
- [ ] `wait_for_load_state(state)` / `wait_for_url(url)`
- [x] `add_script_tag()` / `add_style_tag()`
- [ ] `name` / `page` / `parent_frame` / `child_frames` / `is_detached()` properties

### Page: Event System Expansion

Currently limited to `on_download`, `on_dialog`, `on_websocket`, `on_request`, `on_response`, `on_request_finished`, `on_request_failed`, `pause`. Need:

- [ ] `expect_download()` — Wait for download to start
- [ ] `expect_response(url)` / `expect_request(url)` — Wait for network
- [ ] `expect_popup()` — Wait for popup window
- [ ] `expect_file_chooser()` — Wait for file chooser
- [ ] `expect_console_message()` — Wait for console output
- [ ] `expect_event(event)` — Generic event waiting

### Page: Additional Methods

- [x] `pdf()` — Generate PDF (Chromium only)
- [x] `set_extra_http_headers(headers)`
- [x] `emulate_media()` — CSS media type and color scheme
- [ ] `expose_function(name, callback)` / `expose_binding(name, callback)`
- [x] `unroute_all()` / `unroute(pattern)` — Remove route handlers
- [x] `add_init_script(script)` — Run JS before page scripts
- [x] `viewport_size` property

### Playwright: Missing Properties

- [ ] `devices` — Device descriptor dictionary for emulation
- [ ] `selectors` — Custom selector engine registration

## Release Plan

### v0.9.0 — "Locator, Response & Request Complete"

Complete the three most user-facing data access classes. Mostly trivial/low RPC wrappers.

- **Locator → 100%**: `dispatch_event()`, `bounding_box()`, `scroll_into_view_if_needed()`, `wait_for()`, `drag_to()`, `evaluate()`, `evaluate_all()`, `tap()`, `page` property
- **Response → 100%**: ~~`body()`, `text()`, `json()`, `finished()`, `all_headers()`, `header_value()`, `headers_array()`, `security_details()`, `server_addr()`, `request`/`frame` properties~~ ✅ complete
- **Request → 100%**: ~~`headers`, `post_data`/`post_data_buffer`/`post_data_json`, `failure`, `frame`, `redirected_from`/`redirected_to`, `response()`, `timing`, `all_headers()`, `header_value()`, `headers_array()`, `sizes()`~~ ✅ complete
- **Dialog → 100%**: ~~`page` property (back-reference)~~ ✅ complete
- **Download → 100%**: ~~`page` property (back-reference)~~ ✅ complete
- **Also**: Page assertions (`to_have_title`, `to_have_url`), Response struct field encapsulation (breaking change)

### v0.10.0 — "Frame, FrameLocator & BrowserContext Complete"

Complete frame/iframe support and runtime configuration. Builds timeout infrastructure.

- **Frame → 100%**: Public `locator()`, all 7 `get_by_*`, `frame_locator()`, `add_script_tag()`, `evaluate_handle()`, properties (`name`, `page`, `parent_frame`, `child_frames`, `is_detached`)
- **FrameLocator → 100%**: New class with `locator()`, `frame_locator()`, all 7 `get_by_*`, `owner` property
- **BrowserContext → 100%**: `expect_page()`, `expect_event()`, `expose_function()`, `expose_binding()`, `new_cdp_session()`, `tracing`, `set_default_timeout()`, `set_default_navigation_timeout()`, all `on()` events
- **Also**: Page timeout methods, `is_closed`, `frames`, `frame_locator()`

### v0.11.0 — "Page, Browser & New Classes Complete"

Complete the event system and remaining classes.

- **Page → 100%**: All `expect_*` methods, `expose_function()`/`expose_binding()`, all `on()` events (close, console, crash, filechooser, load, pageerror, popup, frame events, worker)
- **Browser → 100%**: `contexts`, `browser_type`, `start_tracing()`/`stop_tracing()`, `on("disconnected")`
- **FileChooser → 100%**: New class (`element`, `is_multiple`, `page`, `set_files()`)
- **ConsoleMessage → 100%**: New class (`text`, `type`, `args`, `location`, `page`)
- **Selectors → 100%**: `register()`, `set_test_id_attribute()`

### v1.0.0 — "Polish, Docs & Validation"

No new API surface. Folio integration validation, documentation completeness, examples, performance optimization. WebSocket deferred to v1.1.0 as planned.
